🤖 Day 9 Plan — Predictive AI Model

Model goal: Predict a student's score given their study hours, AI tool usage, and subject.


Algorithm: Linear Regression (scikit-learn) — simple, interpretable, appropriate for this dataset size.


Features (inputs):

  • study_hours (numeric)
  • ai_tool_used (boolean → 0/1)
  • subject (one-hot encoded)

Target (output): score (0–100)


What gets built:

  • Train/test split → train model → evaluate (R², MAE)
  • Save model with joblib
  • POST /api/predict endpoint
  • "Predict My Score" widget on the dashboard

Fallback if not built: Document approach in README as "future work" — the dashboard is still a complete project.

🔬 How the Model Works

Algorithm: Linear Regression (scikit-learn)

Features used to predict score:

  • Study hours per week (numeric)
  • AI tool used (Yes=1, No=0)
  • Subject (one-hot encoded — each subject becomes a separate feature)

Training: 80% of uploaded records used to train, 20% held out to test accuracy.

Metrics explained:

  • R² — How well the model explains score variation (0 = useless, 1 = perfect)
  • MAE — Average error in score points (lower = better)
  • RMSE — Like MAE but penalises large errors more heavily
